# Rainblur - A bold, dark themed landing page with personality
#### Preview

 - [Demo](https://themewagon.github.io/Rainblur-Landing-Page/)

#### Download
 - [Download from ThemeWagon](https://themewagon.com/themes/Rainblur-Landing-Page/)

## Getting Started

1. Clone Repository
```
git clone https://github.com/themewagon/Rainblur-Landing-Page.git
```
2. Install Dependencies
```
npm i
```
3. Run the development server:

```bash
npm serve .
```
